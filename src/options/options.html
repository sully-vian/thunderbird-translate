Privacy notice: When you request a translation we send only the email message body (the HTML source or, if not available, the plain-text body) to Google’s Generative Language API (Gemini) to produce the translation. We do not send your account credentials, mailbox headers, recipients, subject, attachments, or other mailbox metadata. The API key you enter is stored locally in the add‑on (browser.storage.local) and is used solely to authenticate requests to Google. HTML markup is included so structure can be preserved; URLs and email addresses are part of the sent content but are instructed not to be altered. Do not translate sensitive or confidential content if you require full privacy. For more information, see the Gemini Apps Privacy Hub: https://support.google.com/gemini/answer/13594961
